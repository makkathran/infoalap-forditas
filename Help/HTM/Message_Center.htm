Message Center

The Message Center is your hub for timely announcements from Freedom Scientific to help maximize your experience with JAWS. The messages cover various topics, including:

  • Product-specific training tips to explore key features.
  • Information about new features and changes in updates.
  • Details about upcoming webinars and additional training opportunities.
  • Notifications about new FSCast podcast episodes.

By default, a notification is spoken and displayed when new messages are available. This is a standard Windows background notification, similar to those received from other apps such as Outlook. To access your messages, do one of the following:

  • Choose Message Center from the Help menu. This menu item also includes the number of unread messages.
  • Use the layered keystroke, INSERT+SPACEBAR, followed by SHIFT+M.
  • Use the Voice Assistant. For example, "Hey Sharky, open message center."

When Message Center opens, you are placed in the list of messages. Use UP or DOWN ARROW to move through the list and press ENTER on a message to view additional details. For example, release notes for an update, step-by-step instructions for a training tip, or an overview of an FSCast episode. If the message details include links to external content, pressing ENTER opens the resource in the appropriate app on your computer. For instance, links to web pages open in your default web browser. Once you are finished reviewing details for a message, press ESC to return to the list.

To filter the list to only show specific messages, press SHIFT+TAB to move to the Search edit field, type what you want to search for, and then press TAB to move back to the list which now displays your search results. For example, entering "podcast" shows only messages related to the Freedom Scientific FSCast podcast.

To delete messages you no longer want, navigate to a message and press the DELETE key or select the Delete button.

To configure Message Center notifications as well as what types of messages are included in the list, select the Settings button. This dialog box contains several check boxes. Clear the Receive notifications about new messages check box if you do not want to receive automatic notifications when new messages are available. The remaining check boxes are the various types of messages that are included in the list. Select or clear the appropriate options depending on what messages you want to receive. All message types are enabled by default. Once you are finished making your selections, select OK to close the dialog box.

To completely close Message Center, press ESC or select the Close button.
