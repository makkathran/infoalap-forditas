Rendszerajánlások

A rendszerajánlások a JAWS-hoz az alábbiakban találhatók.

Leírás Megnevezés
Operációs rendszer (64 bites) Windows® 11, Windows 10, Windows Server® 2025, Windows Server 2022, Windows Server 2019 és Windows Server 2016
A JAWS minden operációs rendszer esetében támogatja az x64 architektúrát.
A JAWS további támogatást nyújt az ARM64 architektúrához a Windows 11 operációs rendszereken.
Processzor sebesség 2.0 GHz-es, kétmagos i5 vagy gyorsabb processzor ajánlott
Memória (RAM) 16 GB vagy több ajánlott
Szabad lemezterület 2 GB, további hangokhoz több hely szükséges
SSD meghajtó ajánlott
Videoadapter Video/grafikus kártya DirectX 9 vagy újabb támogatással és WDDM 1.0 illesztőprogrammal
1024x768 felbontású kijelző
Hangkártya Windows kompatibilis hangkártya (a beszédhez)
