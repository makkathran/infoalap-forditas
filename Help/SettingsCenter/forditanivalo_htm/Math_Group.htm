Math Group

Use the options in this group to configure how JAWS reads math content.

Speech

Use the options in this group to determine how much information JAWS speaks when reading math expressions.

Speech Style

Use the radio buttons in this group to set the speech style for math content.

Select ClearSpeak to hear the full expression with all symbols and terms spoken. For example, you might enable this setting when taking an exam or completing a class assignment .

Selecting SimpleSpeak attempts to streamline speech by omitting the announcement of symbols such as brackets whenever possible. This is the default setting.

Math Verbosity

Use the radio buttons in this group to control verbosity for math. You can choose between Terse, Medium, and Verbose.

For example: the square root of x is spoken as "square root of x" when Verbose is selected, and "square root x" when Terse is selected.

Braille

Use the options in this group to specify how math content is displayed and highlighted in Braille.

Braille Navigation Highlight

Use the radio buttons in this group to control how the currently selected math element is highlighted in Braille.

  • Off - No highlight is displayed.
  • First Character - Only the first character is highlighted. This is the default setting.
  • Endpoints The highlight is on the first and last characters of the selected element.
  • All The entire element is highlighted.

Braille Math Code

Use this combo box to choose the Braille math code to use when displaying math content. Nemeth is selected by default. You can also switch between available Braille math codes by pressing INSERT+EQUALS. Focus display users can also press RIGHT SHIFT+DOTS 1-2-3-4-5-6.

Navigation

Use the options in this group to configure settings for navigating math content.

Navigation Mode

Use the radio buttons in this group to control the navigation mode for math.

  • Enhanced - Moves between mathematically meaningful pieces such as operators and delimiters. This is the default setting.
  • Simple - Moves between words, except when an expression such as a fraction or square root is encountered. In this case, the entire expression is read.
  • Character - Moves between words but allows you to move between characters by moving one level down.

Navigation Verbosity

Use the radio buttons in this group to control how much JAWS speaks as you move between elements in a math expression. You can choose between Terse, Medium (selected by default), and Verbose.

Zoom Out of Expressions Automatically

Select this checkbox to automatically move out of mathematical expressions when you've reached their edge while navigating. This check box is cleared by default.
