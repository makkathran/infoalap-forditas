Szokásos Windows billentyűparancsok

Ez a fejezet a Windows rendszerben használható billentyűkombinációkat ismerteti. A JAWS bizonyos billentyűkombinációknál ki is mondja, hogy mi történik, de ezek a billentyűkombinációk a JAWS futtatása nélkül is működnek. A felsorolás nem a teljesség igényével készült, de tartalmazza a Windows rendszerben használható legfontosabb billentyűkombinációkat.
