A Windows általános billentyűparancsai

Leírás Billentyűparancs
Súgó megnyitása F1
Start menü megnyitása WINDOWS BILLENTYŰ vagy CTRL+ESC
Váltás a nyitott alkalmazások között ALT+TAB
Helyi menü megnyitása HELYI MENÜ vagy SHIFT+F10
Minden alkalmazás kis méretűvé tétele WINDOWS+M
Lépés a tálca első elemére WINDOWS+TAB
Windows Intéző megnyitása WINDOWS+E
Futtatás párbeszédpanel megnyitása WINDOWS+R
